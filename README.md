# Praveen Portfolio (Next.js + Tailwind, App Router)

## Quickstart
```bash
npm i
npm run dev
# open http://localhost:3000
```

## Drop your assets
- Profile photo: `public/assets/profile.jpg`
- Résumé PDF:   `public/assets/PraveenHari_Resume.pdf`  ← linked in nav + hero
- Project covers (ids: dizzitup, robotgames, patient-iot, cmxi, campustrade, environment):
  `public/assets/projects/<id>.jpg`
- Optional gallery: `public/assets/projects/<id>-1.jpg`, `<id>-2.jpg`

If images are missing, a gradient placeholder renders so the page never breaks.
